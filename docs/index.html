Analyze Mode

Inspect all metadata hidden in your PDFs — author, creation date, software used, and more. Export it to a text file for your records.

  • View all metadata fields
  • Export to text file
  • Non-destructive inspection

Clean Mode

Wipe all metadata from your PDF and save a pristine copy. Perfect for sharing documents without revealing your digital footprint.

  • Complete metadata removal
  • Creates clean copy (*-clean.pdf)
  • Original file untouched
